This project is a Phonebook Management System that allows users to store and manage contact information. The system uses a CSV file as a simple database to store names and phone numbers. The project provides basic CRUD operations that enable the user to:

Add new contacts

Update existing contact details

Delete contacts from the phonebook

Display all saved contacts

The program loads contact data from the CSV file when it starts, and updates the file whenever changes are made. The user interacts with the system through a simple menu-driven interface, making it easy to perform operations.

Key Features

Stores contacts in contacts.csv

Loads contacts automatically when the system starts

Allows adding, editing, deleting, and viewing contacts

Ensures changes are saved back to the file

Simple and user-friendly console interface
